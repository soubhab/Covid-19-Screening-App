Data Collection for Covid-19 Screening Application

Pilot data collection for covid-19 screening application development by IIT Kharagpur. This application is a part under Covid-Rakshak, a Central Goverment intiative.

In this application various meta-data, physiological-data and cough sound data will be recorded remotely from Covid positive subjects.
